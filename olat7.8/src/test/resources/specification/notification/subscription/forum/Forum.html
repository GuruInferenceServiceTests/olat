Forum abonnieren

User Story

Als registrierter Benutzer will ich ein Forum abonnieren,
damit ich zu dem von mir gewählten Zeitpunkt über neue Einträge und Änderungen via OLAT Startseite, RSS, E-Mail informiert werde
und somit auf dem neuesten Stand bin.

Akzeptanzkriterien

Automatische Tests

Bemerkungen zu den automat. Tests

  • Falls ein Beitrag ein Zitat enthält, wird das Zitat in der sofortigen Zustellung des E-Mails nicht angezeigt.
  • Der Benutzer soll nicht über eigene Änderungen benachrichtigt werden.
  • E-Mail Inhalt ist HTML-formatiert. Multimedia Internet Mail Extension (MIME) muss unterstützt werden
  • Die tägliche Benachrichtigung wird um 00:10 Uhr verschickt. Sollte dieser Zeitpunkt ungünstig sein aufgrund von Backup und Downzeit von OLAT, muss ein neuer Zeitpunkt gewählt werden.
  • Die Adresse noreply@olat.uzh.ch als Systemadresse muss allenfalls eingerichtet werden, falls noch nicht vorhanden.

Manuelle Tests

  • [.] Benutzerin Berta beantwortet einen Forumsbeitrag "Prüfungsfragen" von Alphonse, waehrendessen der Benutzer Alphonse seinen Beitrag löscht.

Out-of-Scope

  • Ein Attachement zum Forumsbeitrag wird nicht als Attachement im E-Mail Kanal an den Benutzer versendet. Grund:Skalierbarkeit.
