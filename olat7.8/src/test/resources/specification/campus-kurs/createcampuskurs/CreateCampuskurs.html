Campuskurs erstellen

User Story

Als Dozent möchte ich einen OLAT-Kurs erstellen und dabei auf die Informationen auf SAP zurückgreifen können, sodass ich die Info nicht doppelt eingeben muss.

  • Kurs-Template

Hinweise

  • Name Lernbereich: Campuslernbereich
  • Beschreibung: Dieser Lernbereich enthält zwei Campusgruppen A und B.
  • Name SAP-Gruppe für Teilnehmer: Campusgruppe A
  • Beschreibung: Diese Gruppe wird täglich mit dem Modulbuchungssystem abgeglichen. Sie ist daher nicht editierbar.
  • Name Gruppe für externe (nicht via Modulbuchung erfasst): Campusgruppe B
  • Beschreibung: Diese Gruppe wird von den DozentInnen oder BetreuerInnen verwaltet.

GUI-Vorlage Titel und Beschreibung

Vorschlag

  • Campuskurs Semester Jahr
  • Dozierende: Vorname Name des Dozenten, Vorname Name des Dozenten
  • Lehrveranstaltungsinhalt: Lehrveranstaltungsinhalt

Die Inhalte (für Semester Jahr, namen der Dozierenden und Lehrveranstaltungsinhalt) kommen aus SAP. Falls die Felder keine Inhalte haben, wird nichts angezeigt.
