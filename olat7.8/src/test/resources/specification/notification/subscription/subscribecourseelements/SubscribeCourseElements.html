Kursbausteine abonnieren

User Story

Als Dozent/Tutor kann ich einzelne Kursbausteine abonnieren, damit ich über die für mich wichtigen Informationen benachrichtigt werde.

Akzeptanzkriterien

  • Neu ist die Abonnier-Funktion per Button nicht mehr per Link definiert
  • Neu hat die Abonnier- Funktion eine neue Beschriftung: Benachrichtigungen bestellen bzw. Benachrichtigungen abbestellen
  • Wenn man über den Benachrichtigungs-Button fährt sollte es ein Hover/Mouse-Over geben: Benachrichtigungen für Forum bestellen bzw. Benachrichtigungen für Forum abbestellen; Benachrichtigungen für Blog bestellen etc.
  • Neu soll es einen Gruppierungsrahmen geben, damit der Benutzer erkennt, welche Funktionalitäten zusammengehören

Spezifikation Abonnierbutton als PDF

Das Forum innerhalb des Wikis wird in einer späteren Iteration abonnierbar gemacht.

Formatvorlage E-Mail, Zustellung: täglich, bei mehreren Änderungen

From: noreply@olat.uzh.ch
To: Alf@uzh.ch
Subject:[OLAT-Benachrichtigung]: Tägliche Zusammenfassung
